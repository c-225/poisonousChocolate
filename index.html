Comment jouer ?

Bienvenue dans le Jeu du chocolat empoisonné ! C'est un jeu à deux joueurs qui fonctionne ainsi :

  • Poison : Parmi les carreaux de chocolat, il y en a un qui est empoisonné.
  • Objectif : Découper le chocolat horizontalement ou verticalement pour laisser votre adversaire avec le carreau empoisonné.

Utilisez les contrôles pour affronter une IA, ou jouer contre un ami, vous pouvez aussi agrandir ou réduire le chocolat en utilisant le bouton "redimensionner".
